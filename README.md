# 📚 Portfólio — Engenharia de Software | FIAP 2026

## Sobre este repositório

Este repositório reúne as atividades práticas desenvolvidas na disciplina de Engenharia de Software da FIAP durante 1° semestre do ano de 2026.

Os exercícios abordam conceitos fundamentais como:
- Requisitos funcionais e não-funcionais;
- Documento SRS;
- UML;
- Diagramas de atividades, sequência e classes;
- Arquitetura em camadas e MVC;
- Implementação em Python.

As aulas contém diagramas, código-fonte e exemplos de execução desenvolvidos no Draw.io, Miro e Figma e centralizados no Google Colab.

---

# Como executar os exercícios

## Pré-requisitos

- Conta Google

## Instalação

1) Acesse o repositório pelo Link:

```https://github.com/nicolasaalves/Checkpoint-3---Engenharia-de-Software.git```

2) Acesse o arquivo:

Dentro do repositório acesse o arquivo da aula desejada clicando no nome.

Exemplo:
Checkpoint-3---Engenharia-de-Software/Aula_03_(04_03).ipynb

3) Acesse o Google Colab:

Dentro do arquivo, clique em "Open in Colab". Ao ser direcionado para a página do Google Colaboratory faça login com sua conta Google

4) Execução:

Clique em "Executar tudo"
